03 The full truth table
PAMA ∧ MP ∨ (A ∧ M)
Reading the variables:
P — adjusted attendance ≥ 75% (the standard eligibility line)
A — adjusted attendance ≥ 65% (the minimum condonable line)
M — a valid medical certificate has been applied
E — the student is exam-eligible, defined as P ∨ (A ∧ M)
04 As a predicate
For any student x, with Att(x) = adjusted attendance percentage and Med(x) = "x has applied a valid medical certificate":

∀x   Eligible(x) ↔ ( Att(x) ≥ 75 ) ∨ ( Att(x) ≥ 65 ∧ Med(x) )

Same statement as the propositional version above, generalized over every student in the class.
